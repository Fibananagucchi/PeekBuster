# PeekBuster: Cyber-Sentinel for Android

PeekBuster is an advanced Android security and privacy-focused application designed to act as a **Cyber-Sentinel** for your device. It provides real-time monitoring of application behavior, network traffic analysis, and automated permission auditing using high-precision AI.

## 🚀 Key Features

### 🛡️ System Security Overview (Advisor)
The main dashboard provides a high-level summary of your device's security state:
*   **Live Threat Counters**: Real-time indicators for **DANGEROUS** (high-risk) and **SUSPICIOUS** (medium-risk) applications.
*   **System Threat Log**: A live carousel showing the most recent threats identified by the sentinel.
*   **Integrated System Scan**: A high-precision background process that audits every installed app using AI without blocking your UI.

### 🔍 Deep Permission Audit (Audit)
A comprehensive list of all installed apps, strictly sorted by threat level:
*   **Permission Ratios**: Shows exactly how many sensitive permissions are **Granted vs. Requested** (e.g., "2/5 PERMISSIONS").
*   **AI Intel Reports**: Technical explanations generated by AI for each app, explaining *why* certain permissions are requested and if they are legitimate.
*   **Whitelist Management**: Easily manage your "Trusted Apps" through a dedicated management modal.

### 🌐 Real-Time Traffic Shield (Traffic)
A passive VPN-based monitor that logs and analyzes every external connection:
*   **Domain Identification**: Identifies the destination of every data packet.
*   **Automated AI Analysis**: Each connection is analyzed individually by AI to determine if the destination is safe.
*   **Encryption Awareness**: Recognizes standard encryption (TLS/HTTPS) as a positive security practice.

## ⚙️ How It Works: The Sentinel Engine

### Phase 1: Local Heuristics
PeekBuster constantly monitors your device locally using Android system services:
*   **PackageManager**: Scans app manifests for suspicious permission requests.
*   **AppOps & UsageStats**: Tracks real-time usage of the Camera, Microphone, and Location to detect background spying.

### Phase 2: AI-Powered Intelligence
When scanning, PeekBuster sends technical metadata to a high-end AI model via the **Groq API**:
*   **Fairness Directives**: The AI is trained to distinguish between legitimate needs (e.g., Maps needing Location) and suspicious behavior.
*   **Hallucination Protection**: Internal logic cross-checks AI reasoning to prevent false positives if no technical risk is actually identified.

### Phase 3: Passive Traffic Interception
The **Shield Active** mode creates a local VPN tunnel to look at **socket metadata**. No user data is sent to external servers; only destination domains are vetted for reputation.

## 🛠️ Technical Stack
*   **Language**: Kotlin
*   **UI**: Jetpack Compose (Modern Cyber-Sentinel aesthetic)
*   **State**: ViewModel + Coroutines + StateFlow
*   **Networking**: OkHttp + VpnService
*   **AI Engine**: Groq API (LLaMA-3 / Mixtral)

## 🔒 Privacy Commitment
PeekBuster is designed with **Privacy by Design**:
*   **Local Processing**: Your network traffic never leaves your device.
*   **No Cloud Storage**: We do not store your personal data or scan history on our servers.
*   **Transparency**: Every alert is accompanied by a technical "Intel Report" so you understand the "why" behind every security score.
